# Stockify Backend

This is the backend for the Stockify project, built with Node.js.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the backend server:
   ```sh
   npm start
   ```

## Project Structure
- `index.js` - Main entry point
- `package.json` - Project metadata and scripts

## Available Scripts
- `npm start` - Start the backend server
